# Contenido
Este archivo es el resultado de la PEC de laboratorio de Procesadores del Lenguaje.
El objetivo de la práctica era crear un analizador para un lenguaje inventado.

El árbol de directorios es el siguiente:
	- Código: Contiene el código para el árbol que crea el árbol sintáctico,
		manteniendo la tabla de símbolos

	- Gramática.cup: Archivo con la gramática para poder realizar simplemente el
		análisis sintáctico, sin realizar el semántico.

	- PECL1.cup: Archivo con la gramática y el código para crear los nodos del árbol
		y realizar el análisis semántico.

	- PECL1.lex: Archivo de especificación léxica.

	- Pruebas: Directorio con archivos de prueba para demostrar diferentes aspectos
		del lenguaje analizado. Cada archivo especifica los apartados que se
		intentan probar.
